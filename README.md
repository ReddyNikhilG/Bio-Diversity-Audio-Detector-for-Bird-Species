# Bio-Diversity-Audio-Detector-for-Bird-Species

This repository contains the backend and frontend components for the BirdSense AI Audio Species Detector.
- **Backend**: FastAPI server running a model ensemble (BirdNET + YAMNet + Google Perch) with NMF audio separation.
- **Frontend**: Streamlit dashboard UI.
